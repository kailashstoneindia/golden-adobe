Tangerine and sky blue, straight from the two logo lockups, anchored by a warm stone-cream canvas instead of plain white — so the app feels closer to site-visit daylight than a generic e-commerce app.

Foundations

Type system

Poppins for headings and display text — rounded, modern, reads well at all mobile sizes. Inter for all body and UI text — the most legible sans-serif at small sizes on screens. DM Mono for anything numeric: prices, order IDs, points — same family you've already been using for InvoiceGenie, so it stays consistent across your work.
